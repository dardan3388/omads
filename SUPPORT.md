# Support

Use the right channel so OMADS issues end up in the right place.

## Bug Reports And Feature Requests

Please use [GitHub Issues](https://github.com/dardan3388/omads/issues) for:

- bug reports
- feature requests
- documentation problems
- usability feedback

## Security Reports

Please do **not** open public issues for security vulnerabilities.

Use the private reporting instructions in [SECURITY.md](SECURITY.md).

## Commercial Or Collaboration Inquiries

For partnerships, commercial interest, consulting, or other direct inquiries, contact:

- `ceo@shala-mail.de`

Please check:

- [README.md](README.md)
- [docs/getting-started.md](docs/getting-started.md)
